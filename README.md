## Prerequisites

For all installations run:

```shell
bin/setup-mac
```

Note: you need to have your user's pip3 `bin` directory in your `PATH`.
On Mac OS X that is typically `$HOME/Library/Python/3.x/bin`.

## Develop

    $ bin/serve

## Build

    $ bin/build
